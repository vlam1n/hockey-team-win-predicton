# hockey-team-win-predicton
Pet project. 

Цель: Цель проекта постоить модель, которая будет предсказывать победу конкретной хокейной команды (ЦСКА) в предстоящем матче.

Тип задачи: бинарная классификация

Target:
  1 - победа команды
  0 - проигрыш

План реализации:
  1) Сбор данных
  2) Создание признаков
  3) Выбор модели
  4) Тестирование
